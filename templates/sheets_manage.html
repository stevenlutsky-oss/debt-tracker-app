🔧 Setup Instructions

  1. Set up Finta: Go to finta.io to sync your bank accounts to Google Sheets
  2. Get Google Cloud Credentials:
    • Go to Google Cloud Console
    • Create a project and enable the Google Sheets API
    • Create a service account and download the JSON credentials
    • Save the credentials file as credentials.json in the debt-tracker folder
    • Share your Google Sheet with the service account email
  3. Configure the app: Edit sheets_config.py with your spreadsheet ID
